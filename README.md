# Worldcup Minigame

H5 2D football AI simulation game built with Vite, React, and TypeScript.

Live site after GitHub Pages deployment:

https://dudachun.github.io/worldcup-minigame/

## Local Development

```bash
npm install
npm run dev -- --port 5175
```

## Verification

```bash
npm test
npm run build
npm run qa:ui
```

## Deployment

Push to `main`. GitHub Actions builds the static site and deploys `dist` to GitHub Pages.
